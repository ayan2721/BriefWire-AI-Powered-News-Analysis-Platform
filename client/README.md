# BriefWire Frontend

## Setup

1. Copy `.env.example` to `.env`.
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`

## Notes

- The app uses Vite, Tailwind CSS, React Router, React Query, and Framer Motion.
- API requests are proxied to `/api` during development.
- Add `VITE_API_BASE_URL` to point to your deployed backend.
